# Двухмасштабная робастная модель оптимизации портфеля на базе меры риска CVaR с L1-регуляризацией

Программный комплекс на языке Python, разработанный в рамках курсовой работы на Механико-математическом факультете СГУ им. Чернышевского (Кафедра дифференциальных уравнений и математической экономики).

## 🚀 Ключевой функционал
* **Эконометрический анализ:** Сбор, очистка и расчет высших статистических моментов (асимметрии и эксцесса) для 17 макро-активов Московской биржи (MOEX).
* **Линеаризация Рокафеллара–Урясьева:** Сведение невыпуклой оптимизации квантильного риска к задаче линейного программирования высокой размерности ($N+1+T$).
* **Двухканальная робастность:** Синхронная скаляризация тактического (4 месяца) и стратегического (12 месяцев) окон сценариев для защиты от эффекта переобучения.
* **L1-регуляризация:** Внедрение нелинейного транзакционного штрафа за оборот капитала ($\gamma_{cost} = 0.1\%$) в качестве математического якоря против избыточной ребалансировки портфеля.

## 📊 Структура репозитория
* `backtesting_engine.py` — основной вычислительный модуль, содержащий алгоритмы оптимизации (SLSQP / Линейное программирование) и пайплайн скользящего бэктестинга *Out-of-Sample*.
* `statistical_summary_clean.csv` — рассчитанная матрица высших статистических моментов и лептокуртических шоков рынка РФ за 2022–2025 гг.
* `/cache` — локальная база очищенных исторических логарифмических доходностей активов Московской биржи.

## 🛠️ Стек технологий
* **Язык:** Python 3.10+
* **Математика и оптимизация:** `scipy.optimize` (алгоритм SLSQP), `numpy` (векторно-матричные операции).
* **Анализ данных:** `pandas` (обработка временных рядов).
